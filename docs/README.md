# 미션 - 숫자 야구 기능 구현 목록
----
## 💡 기능 목록

- 상대방(컴퓨터)이 1~9 까지 서로 다른 임의의 수 3개 선택하는 기능

- 플레이어가 서로 다른 3자리의 수를 입력하는 기능

- 입력된 값을 체크하는 기능

- 입력된 값에 따라 조건에 맞는 문구를 출력 하는 기능
```
  1. 같은 수 / 같은 자리 => 스트라이크
  2. 같은 수 / 다른 자리 => 볼
  3. 같은 수 없음 => 낫싱
```

- 플레이어가 잘못된 값을 입력하였을 경우 예외처리 기능
  -  `throw` 문으로 예외 발생 후 어플리케이션 종료

- 게임 종료 기능 (3스트라이크 또는 예외처리)
```
  1. 게임 종료 후 재시작 또는 완전 종료 기능
  2. 재시작/종료 구분 입력값 => 1 / 2
```
